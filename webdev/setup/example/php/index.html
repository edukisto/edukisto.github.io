PHP

  • Windows
  • Ubuntu 22.04.3
  • Linux
  • macOS

Установите Open Server Panel или XAMPP.

sudo

Команда sudo выполняется в рамках сессии администратора. Вам будет предложено ввести его пароль. Аккуратно наберите его в окне эмулятора терминала и нажмите Enter. Будьте внимательны: символы не отображаются.

  1. Установим пакеты.

    # Внесём репозиторий в список источников для менеджера пакетов apt.
    sudo LC_ALL=C.UTF-8 add-apt-repository ppa:ondrej/php

    # Обновим индекс пакетов.
    sudo apt update

    # Убедимся, что пакеты PHP доступны для установки.
    apt search php8.3

    # Для проверки имитируем установку пакета с показом полных номеров версий.
    sudo apt install -s php8.3

    # Установим пакет.
    sudo apt install php8.3

    # Для проверки выведем номер установленной версии.
    php --version

    php -m

    # Установим расширения.
    sudo apt install \
      php-curl \
      php-mysql \
      php-xml

  2. Заменим файлы инициализациии PHP.

    # php --ini
    wget https://www.php.net/distributions/php-8.3.2.tar.xz
    tar xf php-8.3.2.tar.xz
    cd php-8.3.2
    sudo cp php.ini-development /etc/php/8.3/apache2/php.ini
    sudo cp php.ini-development /etc/php/8.3/cli/php.ini

  3. В строке Loaded configuration file будет указан путь к загруженному конфигурационному файлу PHP. Откроем его на редактирование.

    sudoedit /etc/php/8.3/apache2/php.ini
    sudoedit /etc/php/8.3/cli/php.ini

  4. Убедимся, что следующие директивы не закомментированы и приведены к указанным значениям.

    ; Часовая зона согласно базе данных IANA.
    date.timezone = Europe/Moscow

    ; Сообщения об ошибках выводить в поток stdout (в браузер).
    display_errors = On

    ; Выводить сообщения об ошибках запуска PHP.
    display_startup_errors = On

    ; Докладывать об ошибках, предупреждениях и уведомлениях.
    error_reporting = E_ALL

    ; Сообщения об ошибках форматировать средствами HTML.
    html_errors = On

    ; Вести лог ошибок.
    log_errors = On
Внимание!

Установке PHP предшествует его компиляция и компоновка.

  1. Скопируем с веб­‑страницы https://www.php.net/downloads гиперссылку на архив с исходным кодом последней стабильной версии PHP.

  2. Скомпилируем, скомпонуем и установим PHP.

    wget https://www.php.net/distributions/php-8.3.2.tar.xz

    tar xf php-8.3.2.tar.xz

    cd php-8.3.2

    sudo apt install \
      build-essential \
      libcurl4-gnutls-dev \
      libonig-dev \
      libsqlite3-dev \
      libssl-dev \
      libxml2-dev \
      pkg-config \
      zlib1g-dev

    # ./configure --help

    ./configure \
      --enable-mbstring \
      --with-curl \
      --with-openssl \
      --with-pdo-mysql

    make -j 4

    make test

    sudo make install

    sudo cp php.ini-development /usr/local/lib/php.ini

  3. Откроем его на редактирование.

    sudoedit /usr/local/lib/php.ini

  4. Убедимся, что следующие директивы не закомментированы и приведены к указанным значениям.

    ; Часовая зона согласно базе данных IANA.
    date.timezone = Europe/Moscow

Установите XAMPP.
