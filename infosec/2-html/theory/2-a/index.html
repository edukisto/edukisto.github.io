Гиперссылка

Элемент a с атрибутом href реализует гиперссылку. В качестве значения атрибута href указывают абсолютный или относительный веб­‑адрес (URI). Между стартовым тегом <a> и конечным тегом </a> размещают текст или некоторые другие элементы HTML (например, img).

<a href="/catalog/">Перейти в раздел «Каталог»</a>
<a href="https://mpt.ru/">Перейти на сайт МПТ</a>
<a href="https://mpt.ru/">
  <img src="mpt-logo.svg" alt="Эмблема МПТ">
</a>

Проектирование​

Советы по проектированию URI:

  • записывайте слова на английском языке;
  • используйте только строчны́е буквы английского алфавита (a — z), арабские цифры (0–⁠9);
  • слова разделяйте и отделяйте от чисел дефисоминусами (-);
  • если возможно, путь начинайте с существительного во множественном числе (например, /posts/, /products/, /users/).

В URI не следует включать:

  • информацию об авторе (у документа может быть несколько соавторов);
  • расширение имени файла.

Постарайтесь спроектировать такой URI, который не потребуется изменять никогда.

Учтите также, что адресной строкой браузера можно пользоваться как средством навигации по веб­‑сайту — пользователь может вручную отредактировать URI и подать по нему запрос. Рассмотрим примеры URI:

  • https://blog.example.com/posts/2023/12/31/x/ — веб­‑страница с постом с заголовком «X» за 31.12.2023;
  • https://blog.example.com/posts/2023/12/31/ — веб­‑страница с перечнем постов за 31.12.2023;
  • https://blog.example.com/posts/2023/12/ — веб­‑страница с перечнем постов за декабрь 2023 г.;
  • https://blog.example.com/posts/2023/ — веб­‑страница с перечнем постов за 2023 г.;
  • https://blog.example.com/posts/2023/ — веб­‑страница с перечнем всех постов в блоге.

Фрагмент​

Если URI содержит фрагмент — необязательный компонент, к

позволяющий сослаться не на документ целиком, а на конкретную его часть, то браузер стабилизирует полосу прокрутки на уровне того элемента, значение атрибута id которого соответствует идентификатору фрагмента. Например, при нажатии на гиперссылку

<a href="#chapter-6">Перейти к главе VI</a>

будет совершён переход к элементу

<h2 id="chapter-6">Глава VI</h2>.
