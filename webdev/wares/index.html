Не используйте «Блокнот» в Windows до версии 8.1 включительно

Версии «Блокнота», поставляемые с операционной системой Windows до версии 8.1 включительно, при сохранении текстового файла в «кодировке» UTF‑8 вставляют в его начало маркер порядка байтов (BOM) — последовательность ⟨EF16; BB16; BF16⟩, которая, появившись в сценарии PHP, может быть отправлена клиенту раньше контента и тем самым помешает установке заголовков HTTP.

В «Блокнот», поставляемый с Windows 10, внесено исправление — добавлена возможность сохранения файлов в UTF‑8 «без спецификации» (без BOM).

Совет

Пользователям операционной системы Windows рекомендуется установить Open Server Panel или XAMPP, которые содержат MySQL и PHP.
