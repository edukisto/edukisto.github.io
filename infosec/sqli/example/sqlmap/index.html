sqlmap

Запуск CLI​

  • macOS
  • Ubuntu
  • Windows 11

Запустим приложение «Терминал».

Запустим приложение «Терминал» (⎈ Ctrl + ⎇ Alt + T).

Запустим приложение «Командная строка» (не PowerShell!).

Установка Python​

Для эксплуатации sqlmap необходим интерпретатор Python версии 2.6, 2.7 или 3. Чтобы убедиться, что установка необходима, подадим команду на вывод номера версии:

python -V

Если выведен номер подходящей версии, то установка Python не требуется, и мы можем перейти к установке sqlmap.

python или python3?

В некоторых Unix­‑подобных операционных системах вместо команды python следует подавать команду python3.

  • macOS
  • Ubuntu
  • Windows 11
  1. Скачаем дистрибутив.
  2. Запустим установку.
  3. Установим галочку «Add... to Path».

Python предустановлен в Ubuntu.

  1. Скачаем дистрибутив.
  2. Запустим установку.
  3. Установим галочку «Add python.exe to Path».

Установка sqlmap​

  • Git
  • macOS
  • Ubuntu
  • Windows 11

  1. Клонируем репозиторий в каталог sqlmap.

    git clone --depth=1 https://github.com/sqlmapproject/sqlmap.git sqlmap

  2. Перейдём в каталог sqlmap.

    cd sqlmap

  1. Скачаем архив и сохраним его как sqlmap-master.zip.

    curl --location --output sqlmap-master.zip -- https://github.com/sqlmapproject/sqlmap/archive/refs/heads/master.zip

    (Опция --location позволяет следовать серверным перенаправлениям.)

  2. Распакуем архив.

    tar -xf sqlmap-master.zip

    Появится каталог sqlmap-master.

  3. Переименуем каталог sqlmap-master в sqlmap.

    mv sqlmap-master sqlmap

  4. Перейдём в каталог sqlmap.

    cd sqlmap

  1. Скачаем архив и сохраним его как sqlmap-master.zip.

    wget -O sqlmap-master.zip https://github.com/sqlmapproject/sqlmap/archive/refs/heads/master.zip

    (Опция -O позволяет указать путь к выходному файлу.)

  2. Распакуем архив.

    unzip -q sqlmap-master.zip

    Появится каталог sqlmap-master.

  3. Переименуем каталог sqlmap-master в sqlmap.

    mv sqlmap-master sqlmap

  4. Перейдём в каталог sqlmap.

    cd sqlmap

  1. Скачаем архив и сохраним его как sqlmap-master.zip.

    curl --location --output sqlmap-master.zip -- https://github.com/sqlmapproject/sqlmap/archive/refs/heads/master.zip

    (Опция --location позволяет следовать серверным перенаправлениям.)

  2. Распакуем архив.

    tar -xf sqlmap-master.zip

    Появится каталог sqlmap-master.

  3. Переименуем каталог sqlmap-master в sqlmap.

    rename sqlmap-master sqlmap

  4. Перейдём в каталог sqlmap.

    cd sqlmap

Использование​

  1. Убедимся, что установлена актуальная версия sqlmap.

    python sqlmap.py --version

  2. Выберем веб­‑страницу, подходящую для тестирования:

    Не все веб­‑ресурсы одинаково полезны

    Успех SQL­‑инъекции зависит от того, такое предложение SQL атаковано.

    1. если целью атаки является сбор сведений о БД или похищение данных, то целесообразно атаковать веб­‑ресурсы, передающие СУБД аргументы для выполнения SQL­‑запросов с SELECT... WHERE... LIMIT... OFFSET. В частности, это веб­‑страницы, на которых выведена карточка товара в интернет­‑магазине, запись в блоге, результаты поиска, фильтрации или постраничного вывода. В URI таких веб­‑ресурсов в том или ином виде фигурируют идентификаторы (?page=42, /products/1337/, /posts/hello-world/ и т. п.).

      Чтобы протестировать такой веб­‑ресурс, скопируйте его URI из адресной строки браузера и передайте sqlmap через опцию --url. Значение идентификатора следует отметить, поставив после него астери́ск (*). Примеры:

      python sqlmap.py --fresh-queries --url=https://shop.example.com/products/?page=42*
      python sqlmap.py --fresh-queries --url=https://shop.example.com/products/1337*/reviews/
      python sqlmap.py --fresh-queries --url=https://blog.example.com/articles/?filter[author.name]=John*

      Если уязвимость обнаружена, будет выведено сообщение:

      URI parameter... is vulnerable.

    2. если целью атаки является добавление, модификация или уничтожение данных, то следует атаковать веб­‑ресурсы, содержащие поля ввода, т. е. элементы HTML <input> или <textarea>, поскольку значения таких полей зачастую служат аргументами для SQL­‑запросов с INSERT, UPDATE, DELETE.

      Флаг --forms обяжет sqlmap осуществить SQL­‑инъекцию в поля ввода, обнаруженные на веб­‑странице с указанным URI.

      python sqlmap.py --forms --fresh-queries --url=http://shop.example/products/edit.php?id=42
      python sqlmap.py --forms --fresh-queries --url=https://blog.example.com/articles/new/

      Если уязвимость обнаружена, будет выведено сообщение:

      POST parameter... is vulnerable.

    3. если целью атаки является предварительный обзор, то можно обязать sqlmap собрать URI потенциально уязвимых веб­‑ресурсов, перемещаясь по гиперссылкам на указанную в опции --crawl глубину начиная с указанного в --url местоположения:

      python sqlmap.py --crawl=4 --fresh-queries --url=http://shop.example/

    4. если необходимо провести инъекцию без автоматизации, то понадобятся опции --data, --url и --method (если ресурс получает данные методом HTTP, отличным от GET).

    В следующем примере веб­‑ресурсу методом POST передаются переменные title и price, равные строкам «Молоко» и «123» соответственно.

    python sqlmap.py --data="title=Молоко&price=123" --fresh-queries --method=POST --url=http://shop.example/products/store.php

    В следующем примере происходит вставка значения ' OR 1 = 1 -- в поле username.

    python sqlmap.py --data="username=' OR 1 = 1 -- " --fresh-queries --method=POST --url=http://shop.example/auth/

    Опции sqlmap можно найти в полной справке к программе (опция -hh). Особый интерес представляет раздел Enumeration.

    python sqlmap.py -hh

    Так, опции -D, -T, -U позволяют задать имя БД, таблицы и пользователя соответственно, если, конечно, они известны пен­‑тестеру. Комбинируя их с другими опциями, можно выбрать все таблицы указанной БД, все кортежи из указанной таблицы указанной БД и т. п.
