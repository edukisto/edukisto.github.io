Задание № 3​

В DVWA составить фрагмент предложения SQL, позволяющий посредством вызова функции вывести заданное значение. Номер варианта соответствует номеру студента в журнале успеваемости, поделённому без остатка на 5 (например, 185=3\frac{18}{5} = 3518​=3).

  1. Перечень ролей (обычно NONE).
  2. Имена аутентифицированного пользователя и хоста.
  3. Номер версии СУБД (например, 8.0.35).
  4. Имена пользователя и хоста, использованные клиентом при подключении к серверу СУБД (root@localhost).
  5. Имя схемы БД (dvwa).

Задание № 4​

Составить консольные команды, которые посредством sqlmap позволяют...

  1. ...извлечь имя пользователя СУБД, от имени которого выполняются запросы с веб­‑сайта.
  2. ...извлечь имя используемой БД.
  3. Извлечь имена таблиц в заданной БД.

Задание № 5​

Составить консольную команду, которая посредством sqlmap позволяет...

  1. ...извлечь версию (баннер) СУБД.
  2. ...извлечь имя хоста.
  3. ...узнать, является ли текущий пользователь БД администратором.
  4. ...перечислить имена всех пользователей БД.
  5. ...перечислить имена баз данных.
  6. ...перечислить привилегии текущего пользователя.
  7. ...перечислить роли текущего пользователя.
  8. ...получить описание атрибутов заданной таблицы *.
  9. ...число записей в таблице БД *.
  10. ...извлечь все кортежи из заданной таблицы *.

* — задание повышенной сложности.

Задание № 6​

Проверить исходный код своего индивидуального проекта на наличие уязвимостей к SQL­‑инъекциям. Обычно они возникают вследствие конкатенации или интерполяции строк, содержащих предложения SQL и их аргументы.

$query = 'SELECT * FROM products WHERE id = ' . $_GET['id'];
$query = "SELECT * FROM products WHERE id = {$data->id}";
$query = "SELECT * FROM products WHERE id = $_GET[id]";
$query = "SELECT * FROM products WHERE id = $id";
