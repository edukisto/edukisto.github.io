Пример выполнения задания

Проект на Laravel​

  1. Откроем эмулятор консоли.

  2. Перейдём в каталог веб­‑сервера (/htdocs в XAMPP, domains/localhost в Open Server Panel).

  3. Создадим шаблон веб­‑приложения на каркасе Laravel.

    laravel new shop

    Если команда не найдена, скачайте архив с шаблоном проекта, распакуйте его в вышеназванный каталог.

  4. Запустите клиент MySQL в интерактивном режиме:

    mysql -u root -p

    Введите пароль.

  5. Создайте БД shop.

    CREATE DATABASE shop;

    Должно быть выведено:

    Query OK, 1 row affected...

  6. Выйдите из клиента MySQL.

    EXIT;

    Должно быть выведено:

    Bye

  7. Если MySQL использует нестандартные настройки, отредактируем файл .env:

    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=shop
    DB_USERNAME=root
    DB_PASSWORD=

  8. Создадим шаблоны программных модулей для сущности Product.

    php artisan make:model --all --api Product
    php artisan make:resource ProductResource
    php artisan make:resource --collection ProductCollection

  9. Приведите сценарий миграции для таблицы products в соответствие

    database/migrations/YYYY_MM_DD_hhmmss_create_products_table.php
    <?php

    use Illuminate\Database\Migrations\Migration;
    use Illuminate\Database\Schema\Blueprint;
    use Illuminate\Support\Facades\Schema;

    return new class extends Migration
    {
        /**
         * Run the migrations.
         */
        public function up(): void
        {
            Schema::create('products', function (Blueprint $table) {
                $table->id();
                $table->string('name', 55)->collation('utf8mb4_0900_ai_ci')->unique();
                $table->decimal('price', 8, 2, true)->index();
                $table->timestamps();
            });
        }

        /**
         * Reverse the migrations.
         */
        public function down(): void
        {
            Schema::dropIfExists('products');
        }
    };

  10. Выполним миграцию структуры БД.

    php artisan migrate

  11. Если при миграции структуры БД выведены сообщения об ошибках, то перед повторной попыткой отредактируйте все файлы в каталоге database/migrations/ таким образом, чтобы вызовы метода string() получили второй аргумент — натуральное число 191 (это максимальная длина индексируемой строки в старых 3­‑байтовых кодировках MySQL, тогда как string() по умолчанию создаёт строку длиной 255 знаков). Например, вызов

    $table->string('name');

    следует заменить на

    $table->string('name', 191);

  12. Реализуем модель товара.

    app/Models/Product.php
    <?php

    namespace App\Models;

    use Illuminate\Database\Eloquent\Factories\HasFactory;
    use Illuminate\Database\Eloquent\Model;

    class Product extends Model
    {
        use HasFactory;

        protected $fillable = [
            'name',
            'price',
        ];
    }

  13. Реализуем методы классов, обеспечивающих контроль ввода данных.

    app/Http/Requests/StoreProductRequest.php
    <?php

    namespace App\Http\Requests;

    use Illuminate\Foundation\Http\FormRequest;

    class StoreProductRequest extends FormRequest
    {
        /**
         * Determine if the user is authorized to make this request.
         */
        public function authorize(): bool
        {
            return true;
        }

        /**
         * Get the validation rules that apply to the request.
         *
         * @return array<string, \Illuminate\Contracts\Validation\ValidationRule|array<mixed>|string>
         */
        public function rules(): array
        {
            return [
                'name' => [
                    'max:55',
                    'required',
                    'string',
                    'unique:products,name',
                ],
                'price' => [
                    'decimal:0,2',
                    'max:999999.99',
                    'min:0',
                    'required',
                ],
            ];
        }
    }
    app/Http/Requests/UpdateProductRequest.php
    <?php

  14. Реализуем методы контроллера.

    app/Http/Controllers/ProductController.php
    <?php

    namespace App\Http\Controllers;

    use App\Http\Requests\StoreProductRequest;
    use App\Http\Requests\UpdateProductRequest;
    use App\Http\Resources\ProductCollection;
    use App\Http\Resources\ProductResource;
    use App\Models\Product;

    class ProductController extends Controller
    {
        /**
         * Display a listing of the resource.
         */
        public function index()
        {
            $products = Product::all();
            return new ProductCollection($products);
        }

        /**
         * Store a newly created resource in storage.
         */
        public function store(StoreProductRequest $request)
        {
            $data = $request->validated();
            $product = Product::create($data);
            return response()->noContent(201)->withHeaders([
                'Location' => route('products.show', $product->id),
            ]);
        }

        /**
         * Display the specified resource.
         */
        public function show(Product $product)
        {
            return new ProductResource($product);
        }

        /**
         * Update the specified resource in storage.
         */
        public function update(UpdateProductRequest $request, Product $product)
        {
            $data = $request->validated();
            $product = $product->update($data);
            return response()->noContent(204);
        }

        /**
         * Remove the specified resource from storage.
         */
        public function destroy(Product $product)
        {
            $product->delete();
            return response()->noContent(204);
        }
    }

  15. Создадим группу веб­‑адресов и свяжем их с методами контроллера.

    routes/api.php

    use Illuminate\Http\Request;
    use Illuminate\Support\Facades\Route;

    |--------------------------------------------------------------------------

    |

    */

    Route::middleware('auth:sanctum')->get('/user', function (Request $request) {
        return $request->user();
    });

    Route::resource('products', ProductController::class);

  16. Выведем перечень маршрутов:

    php artisan route:list

  17. Запустим отладочный веб­‑сервер:

    php artisan serve

    Должно быть выведено сообщение «Server running», содержащее URL, по которому веб­‑сервер ждёт запросов (по умолчанию http://127.0.0.1:8000/).

Проверка API​

Проверим API посредством Postman согласно таблице.

МетодURLСтатус ответа
POSThttp://127.0.0.1:8000/api/products/201
GEThttp://127.0.0.1:8000/api/products/200
GEThttp://127.0.0.1:8000/api/products/ID/200
PATCHhttp://127.0.0.1:8000/api/products/ID/204
DELETEhttp://127.0.0.1:8000/api/products/ID/204
GEThttp://127.0.0.1:8000/api/products/200

Перед подачей запроса на вкладке Headers следует указать:

KeyValue
Acceptapplication/vnd.api+json

При запросах методами POST (добавление) и PATCH (модификация) необходимо также заполнить вкладку Body. Ключи должны соответствовать именам полей в таблице БД.

KeyValue
nameНаименование продукта
priceЦена продукта

Пользовательский интерфейс​

  1. Создадим каталог shop-ui.

  2. Откроем ещё один эмулятор терминала.

  3. Запустим отладочный сервер, встроенный в PHP:

    php -S 127.0.0.2:8080

  4. Создадим файл index.html.

    shop-ui/index.html
    <!DOCTYPE html>
    <html lang="ru-luna1918">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport"
            content="width=device-width, initial-scale=1.0">
      <title>Магазин</title>
      <!-- Подключаем файл со сценарием JavaScript. -->
      <script defer
              src="main.js"
              type="module"></script>
    </head>
    <body>
      <article>
        <h1>Товары</h1>
        <section>
          <h2>Добавление записи</h2>

          <!-- Форма для создания записи о товаре. -->
          <!-- Значения атрибутов name равны именам столбцов в таблице БД. -->
          <form action="http://127.0.0.1:8000/api/products/"
                method="post">
            <label>
              Наименование:
              <input name="name"
                     required
                     value="Молоко">
            </label>
            <label>
              Цена:
              <input min="0.01"
                     name="price"
                     required
                     step="0.01"
                     type="number"
                     value="89.99">
            </label>

            <!-- Для удобства доступа из JavaScript даём кнопке идентификатор. -->
            <button id="store"
                    type="submit">Сохранить</button>
          </form>
        </section>
        <section>
          <h2>Витрина</h2>
          <!-- Для удобства доступа из JavaScript даём контейнеру идентификатор. -->
          <div id="product-list"></div>
        </section>
      </article>
    </body>

    </html>

  5. Создадим файл product-controller.js.

    shop-ui/product-controller.js
    class ProductController {
      async getAll() {
        // URL ресурса API, который принимает запросы на выборку всех записей из БД.
        const url = 'http://127.0.0.1:8000/api/products/';
        // Подаём HTTP-запрос.
        const response = await fetch(url, {
          method: 'GET',
        });
        // Парсим JSON, пришедший в теле ответа сервера.
        const products = await response.json();

        // Создаём пустой контейнер для элементов HTML.
        const articles = document.createDocumentFragment();

        // Перебираем записи о товарах.
        for (const product of products.data) {
          // Создаём контейнер товара.
          const article = document.createElement('article');

          // Создаём заголовок.
          const h1 = document.createElement('h1');

          // Создаём кнопку.
          const button = document.createElement('button');
          button.textContent = 'Удалить';
          // Обработчик щелчка мыши по кнопке.
          button.addEventListener('click', async () => {
            const destroyURL = 'http://127.0.0.1:8000/api/products/' + product.id;
            // Подаём HTTP-запрос.
            const response = await fetch(destroyURL, {
              method: 'DELETE',
            });
            if (response.ok) {
              alert('Запись удалена');
            } else {
              alert('Ошибка!');
            }
          });

          h1.textContent = product.name;
          article.appendChild(h1);
          article.appendChild(button);
          articles.appendChild(article);
        }
        const container = document.getElementById('product-list');
        container.appendChild(articles);
      }

      async store(event) {
        // Отмена стандартного поведения кнопки (отправка формы с переходом).
        event.preventDefault();

        // Копируем данные из формы в объект.
        // Имена полей объекта соответствуют именам столбцов в таблице БД.
        const data = {
          'name': document.getElementsByName('name')[0].value,
          'price': document.getElementsByName('price')[0].value,
        };

        // Отладочная печать в консоль браузера.
        // console.log(JSON.stringify(data));

        // URL ресурса API, который принимает запросы на добавление записей в БД.
        const url = 'http://127.0.0.1:8000/api/products/';

        // Подаём HTTP-запрос.
        const response = await fetch(url, {
          // Кодируем в JSON данные, взятые из формы.
          body: JSON.stringify(data),
          headers: {
            // Сообщаем серверу, что ожидаем ответ в формате JSON.
            'Accept': 'application/vnd.api+json',
            // Сообщаем серверу, что отправляем ему данные в формате JSON.
            'Content-Type': 'application/vnd.api+json',
          },
          method: 'POST',
        });

        // Отладочная печать заголовков ответа в консоль браузера.
        // console.log(response.headers.forEach(console.log));

        if (response.ok) {
          alert('Запись добавлена');
        } else {
          alert('Ошибка!');
        }
      }
    }

    export {
      ProductController,
    };

  6. Создадим файл main.js.

    shop-ui/main.js
    import { ProductController } from './product-controller.js';

    window.addEventListener('load', function () {
      const controller = new ProductController();

      const button = document.getElementById('store');
      button.addEventListener('click', controller.store);

      controller.getAll();
    });

  7. Откроем в браузере веб­‑страницу http://127.0.0.2:8080/.
