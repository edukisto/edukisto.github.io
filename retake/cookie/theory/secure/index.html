Secure

Атрибут Secure гарантирует, что cookie либо будет установлена по защищённом каналу связи (например, по прокотолу HTTP/⁠TLS), либо не будет установлена вовсе.

setcookie('имя', 'значение', [
    'secure' => true,
]);
  • Windows XAMPP

Добавим в конец файла apache\conf\extra\httpd-vhosts.conf:

<VirtualHost shop.example.com:443>
  DocumentRoot "C:/xampp/htdocs/shop.example.com"
  ServerName shop.example.com:443

  SSLEngine on
  SSLCertificateFile "conf/ssl.crt/server.crt"
  SSLCertificateKeyFile "conf/ssl.key/server.key"
</VirtualHost>

В файл C:\Windows\System32\drivers\etc\hosts добавим:

127.0.0.1 shop.example.com

Обратимся по веб­‑адресу https://shop.example.com/.
